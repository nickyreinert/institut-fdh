Primes Web-App

Die Web-App Primes ist eine in JavaScript implementierte Anwendung zur Visualisierung von Primzahlen auf einer spiralförmigen Anordnung. Sie ermöglicht eine interaktive und dynamische Darstellung mathematischer Strukturen durch verschiedene anpassbare Parameter.

Ein zentrales Merkmal der Anwendung ist die Möglichkeit, zahlreiche Konfigurationsparameter individuell zu modifizieren. Dazu gehören beispielsweise die Skalierung der Spiralform, die Farbkodierung der Primzahlen sowie weitere grafische Eigenschaften. Darüber hinaus können viele dieser Parameter animiert werden, wodurch sich zeitabhängige Veränderungen der Visualisierung realisieren lassen.
